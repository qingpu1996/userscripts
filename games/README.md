# Games

Create one folder per game:

```text
games/
  game-name/
    README.md
    game-name.user.js
```

Use lowercase kebab-case for folder names, for example:

```text
antimatter-dimensions/
cookie-clicker/
kittens-game/
```

Each game folder should describe:

- target URL
- script goals
- enabled automation features
- known risks or limits
- manual controls
